# Niver Amorzinho 2025

Este é um site romântico e divertido criado especialmente para o aniversário da minha namorada.

O projeto foi desenvolvido com HTML, CSS e JavaScript puros, sem uso de frameworks ou backend, permitindo hospedagem totalmente estática via GitHub Pages.

## Funcionalidades

- Três minijogos:
  - Jogo da Memória com fotos do casal
  - Quiz Romântico com lógica de acerto/erro
  - Quebra-Cabeça com imagem real
- Cada jogo concede um animalzinho como recompensa
- Após completar todos os jogos, uma mensagem em áudio é desbloqueada
- Não utiliza cookies ou banco de dados, apenas memória de sessão

## Tecnologias

- HTML5, CSS3 e JavaScript (vanilla)
- GitHub Pages para hospedagem
- Live Server para testes locais
- `sessionStorage` para controle de progresso durante a sessão

## Como testar localmente

1. Instale a extensão **Live Server** no Visual Studio Code
2. Abra o arquivo `index.html` com o Live Server
3. Navegue pelos jogos e aproveite a experiência completa
